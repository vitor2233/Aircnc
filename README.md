# Aircnc
Air Coffe and Code is a web app for add spots, code using React.js and Node.js during the Rocketseat Omnistack 9.0

## Installation

### Clone

- Clone this repo to your local machine using `https://github.com/vitor2233/Aircnc.git`

### Setup

> install npm

```shell
$ npm install
$ npm start
```
